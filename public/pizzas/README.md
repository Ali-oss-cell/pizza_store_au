# Pizza Images Directory

Place your pizza images in the following locations:

## Main Hero Images
Place large, high-quality pizza images (recommended: 800x800px or larger) in:
```
public/pizzas/main/
```

**Required files:**
- `mushroom-flex.jpg` (or .png)
- `cheese-burst.jpg` (or .png)
- `pepperoni-heat.jpg` (or .png)
- `garden-fresh.jpg` (or .png)

## Thumbnail Images
Place smaller thumbnail images (recommended: 300x300px) in:
```
public/pizzas/thumbnails/
```

**Required files:**
- `mushroom-flex-thumb.jpg` (or .png)
- `cheese-burst-thumb.jpg` (or .png)
- `pepperoni-heat-thumb.jpg` (or .png)
- `garden-fresh-thumb.jpg` (or .png)

## Image Format
- Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`
- Main images should be square or circular (will be displayed in a circular frame)
- Thumbnails should be square (will be displayed in circular thumbnails)
- Images with transparent backgrounds work best for the pizza itself

## Tips
- Use high-quality images for the main hero section
- Keep thumbnail file sizes smaller for faster loading
- Ensure images are well-lit and show the pizza clearly
- The images will be automatically cropped to fit the circular frames
